# Translator Using Python & Tkinter

This project is a **desktop-based Text Translation and Speech Application** developed using **Python and Tkinter**.  
It allows users to enter text, translate it into multiple languages, listen to the translated speech, and save the audio output.

The application supports **multilingual translation** using the **Google Translator API (deep_translator)** and provides **text-to-speech conversion** using the **pyttsx3 engine**.

## Features
- Text translation between multiple languages  
- Text-to-speech conversion  
- Male and female voice selection  
- Adjustable speech speed  
- PDF text extraction  
- Audio (WAV) to text conversion  
- Download translated speech as audio  

## Technologies Used
- Python  
- Tkinter  
- deep_translator  
- pyttsx3  
- SpeechRecognition  
- PyPDF2  

## Use Cases
- Language learning  
- Educational projects  
- Accessibility support  
- Document translation
